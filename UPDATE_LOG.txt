### How to Run on RDK X5

```bash
# SSH into board
ssh sunrise@192.168.1.185

# Start AI Service
cd /home/sunrise/ai_service
export CAMERA_SOURCE=8
python3 ai_server.py

# Start Backend (new terminal)
cd /home/sunrise/backend
node server.js

# Start Frontend (new terminal)
cd /home/sunrise/frontend
npm run dev -- --host 0.0.0.0
```
